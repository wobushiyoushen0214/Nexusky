# Nexusky — Product Context

## Product Purpose

AI-powered knowledge base and note-taking desktop app. Combines Obsidian-style bidirectional linking with AI-driven semantic search, automatic note association, and conversational retrieval. Local-first, privacy-respecting.

## Users

- Knowledge workers, researchers, writers who maintain large personal knowledge bases
- Developers who want to connect ideas across projects
- Students organizing study materials with AI assistance
- Age range: 22–45, tech-savvy, value efficiency and aesthetics

## Brand & Tone

- **Name**: Nexusky
- **Personality**: Intelligent, calm, focused. A quiet companion that amplifies thinking.
- **Tone**: Minimal, confident, unhurried. Never flashy or attention-seeking.
- **Anti-references**: Notion (too busy/corporate), Evernote (dated), generic note apps (bland)
- **Inspirations**: Linear (precision), Raycast (speed + polish), Arc Browser (bold but restrained), iA Writer (typographic clarity)

## Register

product

## Strategic Principles

1. **Content first**: The editor is the hero. Chrome should disappear when writing.
2. **Depth on demand**: AI features surface only when invoked, never intrusive.
3. **Spatial memory**: Consistent layout so users build muscle memory.
4. **Dark by default**: Optimized for long sessions. Light mode is secondary.
5. **Speed as feature**: Every interaction should feel instant.
