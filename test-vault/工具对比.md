# 工具对比

## 项目管理工具

| 工具 | 适合 | 特点 |
|------|------|------|
| Linear | 工程团队 | 快速、键盘优先 |
| Jira | 大型团队 | 功能全面、可定制 |
| Notion | 小团队 | 灵活、文档+任务 |
| Trello | 简单看板 | 直观、免费 |

## 协作工具

| 工具 | 类型 | 特点 |
|------|------|------|
| Slack | 即时通讯 | 频道组织、集成丰富 |
| Figma | 设计协作 | 实时多人编辑 |
| GitHub | 代码协作 | PR、Issue、Actions |

## 知识管理

| 工具 | 特点 |
|------|------|
| Obsidian | 本地优先、双向链接 |
| Nexusky | AI 驱动、语义搜索 |
| Notion | 在线协作、模板丰富 |

这些工具在 [[项目管理]] 和 [[团队协作]] 中各有用武之地。选择时要考虑团队规模和 [[敏捷开发]] 实践的需求。
